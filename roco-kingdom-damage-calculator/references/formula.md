# 洛克王国PvP属性计算与伤害公式

## 一、属性计算

属性值包含 **生命、物攻、魔攻、物防、魔防、速度**，受 **种族值、个体值、性格修正** 影响。

### 性格修正取值
- `+0.2`（+20% 有利）
- `-0.1`（-10% 不利）
- `0`（无修正）

### 生命值公式
```
生命 = [1.7 × 种族值 + 0.85 × 个体值 + 70] × (1 + 性格修正) + 100
```

### 其他五项（物攻/魔攻/物防/魔防/速度）
```
属性 = [1.1 × 种族值 + 0.55 × 个体值 + 10] × (1 + 性格修正) + 50
```

> 个体值为0到10，且最多只能分配到3个属性上。

## 二、伤害计算

### 详细公式
```
伤害 = 进攻方攻击 ÷ 防御方防御 × 0.9 × (技能威力 × 对应倍率 + 威力加成) × 能力等级 × 威力提升buff × 本系加成 × 克制关系 × 天气影响 × (1 - 减伤)
```

### 能力等级公式
```
能力等级 = (1 + 我方攻击提升 + 敌方防御降低) ÷ (1 + 我方攻击降低 + 敌方防御提升)
```

### 各项说明

| 项目 | 说明 | 默认值 |
|------|------|--------|
| 对应倍率 | 如热身技能使下次攻击翻倍，倍率为2 | 1 |
| 威力加成 | 额外威力数值 | 0 |
| 威力提升buff | 威力倍率提升 | 1 |
| 本系加成 | 技能为本系时1.25，否则1 | 1 |
| 克制关系 | 单克制2倍，双属性都克制3倍，被克制/抵抗0.5倍 | 1 |
| 天气影响 | 雨天水系1.5倍 | 1 |
| 减伤 | 多个减伤效果乘算 | 0 |

### 技能类型与攻防属性对应
- **物攻**技能: 使用进攻方物攻 / 防御方物防
- **魔攻**技能: 使用进攻方魔攻 / 防御方魔抗
- **状态/防御**技能: 不造成伤害

### 克制关系
- 通过防御方属性的 defense.weak / defense.resist 查找
- 双属性防御方: 各自倍率相乘
- 双属性攻击方都克制: 3倍(而非4倍)

### 暴击
- 基础暴击率: 5%
- 暴击伤害: 1.5倍
